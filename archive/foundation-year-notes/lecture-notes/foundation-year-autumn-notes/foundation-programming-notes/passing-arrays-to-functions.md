# Passing Arrays and Strings to Functions

## Method 1

Formal parameters as a pointer −

```text
void myFunction(int *param)
{
   .
   .
   .
}
```

## Method 2

Formal parameters as a sized array −

```text
void myFunction(int param[10])
{
   .
   .
   .
}
```

## Method 3

Formal parameters as an unsized array −

```text
void myFunction(int param[])
{
   .
   .
   .
}
```
